# Medbook AI

A comprehensive Clinic Management System with AI integration. Built with the MERN stack (MongoDB, Express, React, Node).

## Features
- Patient record management.
- Appointment scheduling and calendar view.
- AI Assistant for clinic management support.
- Role-based access control (Admin, Doctor, Nurse, Receptionist).
- Beautiful, dark-themed UI.

## Tech Stack
- **Frontend:** React 18, Vite.
- **Backend:** Node.js, Express, MongoDB (Mongoose).
- **AI:** Google Generative AI (Gemini).

## Setup Instructions

### Backend
1. Navigate to `backend` folder.
2. Run `npm install`.
3. Create a `.env` file and fill in your credentials.
4. Run `npm start` or `npm run dev`.

### Frontend
1. Navigate to `frontend` folder.
2. Run `npm install`.
3. Create a `.env` file and set `VITE_API_URL` to your backend URL if deploying separately.
4. Run `npm run dev` to start the development server.

## Deployment
- **Frontend:** Can be deployed on Vercel or Netlify.
- **Backend:** Can be deployed on Render or Railway.
